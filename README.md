# Experiment 57: Multithreading (Runnable Interface)

## Problem Statement

Create a class `MyRunnable` that implements `Runnable`.
The `run` method should print numbers from 1 to 5.
Create a thread using this runnable and start it.

## Input Format

* No input.

## Output Format

* `1 2 3 4 5` (space or newline separated).

### Example 1

**Input:**

```text
No input
```

**Output:**

```text
1 2 3 4 5
```
